# weeks-we.github.io

Personal academic homepage for `weeks-we`, built as a lightweight static GitHub Pages site.
The layout follows a classic academic profile style: left profile sidebar, right content column,
research topics, education, news, publications, and honors.

## Edit Guide

Most content lives in:

- `index.html`: name, school, bio, research interests, education, news, publications, honors, contact links
- `styles.css`: colors, layout, responsive design
- `assets/avatar-placeholder.svg`: temporary profile photo placeholder
- `assets/publication-figure.svg`: temporary publication figure placeholder

Good first edits:

- Replace `Weeks We` with your real English name or Chinese-English display name.
- Replace `your.email@example.com` with your email address.
- Replace `assets/avatar-placeholder.svg` with your real profile photo.
- Replace the publication placeholder with your real paper, project image, code link, and project page.
- Add scholarships, competitions, papers, awards, or CV links when ready.

## Publish

After editing locally:

```powershell
git add .
git commit -m "Update personal homepage"
git push
```

GitHub Pages will publish the site at:

```text
https://weeks-we.github.io
```
